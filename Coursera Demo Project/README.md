# Overview
Globe Pulse is a demo Flutter application created as part of a Coursera course project submission.
The app demonstrates mobile application development concepts such as API integration, UI design, navigation, and authentication using Flutter and Firebase.

It allows users to explore global news in an intuitive and visually clean interface.

# âœ¨ Features

News API Integration: Fetches latest global headlines.
Category-based News: Technology, Sports, Business, Health, and more.
User Authentication: Login and Signup using Firebase.
Modern UI: Material Design 3 with consistent typography.
Splash Screen: Smooth startup experience.
Bottom Navigation Bar: Easy navigation between sections.
Reusable Widgets: Clean and maintainable code structure.
